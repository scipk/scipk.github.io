AE 303 - Experimental Aerodynamics

Spring 2025 | San Diego State University

CFD MATLAB Wind Tunnel Testing Technical Writing

Overview

This two-unit course focuses on the fundamentals of experimental aerodynamics, including wind tunnel testing, data acquisition, and analysis techniques. The project involved designing and conducting experiments to validate aerodynamic theories and principles learned in other classes.

This class used MATLAB for data analysis and visualization and LaTeX for report writing. It was taught by Dr. Xiaofeng Liu.

GitHub Repository

All LaTeX source code, MATLAB scripts, and project files are available on GitHub. The repository includes the complete LaTeX documentation for all lab reports and data analysis scripts used throughout the course.
